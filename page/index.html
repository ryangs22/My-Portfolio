Sobre Mim

Sou estudante de Engenharia de Computação pela UFAL - Universidade Federal de Alagoas e estou em trajetória para me tornar um Desenvolvedor Full Stack. Minha base técnica é sólida em C , linguagem que me proporcionou um entendimento profundo sobre lógica de programação, estruturas de dados e eficiência computacional. Atualmente, atuo no desenvolvimento front-end com HTML, CSS, Javascript e React, criando interfaces intuitivas, responsivas e centradas na experiência do usuário. Paralelamente, estou ampliando minha expertise para o back-end e análise de dados com Python e Node.js. Mais do que dominar linguagens, meu objetivo é utilizar a tecnologia como uma ferramenta para inovação e solução de problemas reais. Acredito no poder da programação para automatizar tarefas cotidianas, aumentando a eficiência e reduzindo erros operacionais além de desenvolver softwares/aplicações web que simplifiquem processos complexos, conectem pessoas e gerem valor tangível para negócios e para a sociedade. Meu foco atual está no desenvolvimento de sites e softwares, com a ambição futura de expandir meus conhecimentos para o ecossistema mobile, integrando todas as camadas da tecnologia para criar soluções completas e inovadoras para o dia a dia.
